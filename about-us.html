Quem somos

CED é uma organização criada pelo e pertencente ao Grupo B, cujo desejo é aumentar grandemente a facilidade com que se pode adquirir um livro usando as novas tecnologias.

CED é uma sigla para "Celeste, Cristiano, Edvaldo e Délcio". É uma plataforma de livros disponíveis para leitura online e para download gratuito, criada pelo Grupo B, inc. em novembro de 2020 a propósito de um projeto para uma aplicação web.
